Bounce - A Visual Basic project for a ball bouncing.


Links to the various versions:


This project is an attempt to create the Sam Davis sample project from Chapter 8 of his Year 12 text for Software Design & Development course. (NSW HSC syllabus, Australia).

Students are welcome to fork the project to date, but there are many more parts to complete. A little more is done each year when the project comes around.

The repository is at: https://github.com/bwattle/28-Bounce-VB
